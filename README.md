## Run Locally
Clone the project

```bash
git clone https://github.com/Danila0987654/First_3D_Portfolio_Site.git
```

Go to the project directory

```bash
cd First_3D_Portfolio_Site
```

Install packages

```bash
npm install
```

Run a local project

```bash
npm run dev
```
